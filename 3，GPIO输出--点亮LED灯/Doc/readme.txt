【*】程序简介

-工程名称：使用固件库点亮LED灯
-实验平台: 野火STM32-F103 指南者 开发板 
-ST固件库版本：3.5.0

【 ！】功能简介：
控制板子上的LED灯

【 ！】实验操作：
编译并下载程序到开发板，LED灯出现不同颜色的变换

/*********************************************************************************************/


		
【&】程序描述：***bsp为Board Support Packet  (板级支持包）的缩写，每次文件命名都以这个开头是为了后面方便移植***
      <bsp_led.h>:   1.对LED的GPIO的端口、时钟、具体引脚的宏定义
	             2.宏定义高低电平
	             3.设置标准固件库定义带参宏，宏定义LED灯亮灭状态
	             4.设置寄存器方法定义带参宏，并宏定义使用PWM混出的全彩颜色						
       <bsp_key.c>:GPIO的配置：1.定义一个GPIO_InitTypeDef类型的结构体	          
			     2.开启端口外设时钟
			     3.选择要控制的GPIO的引脚
			     4.设置GPIO的模式、速度
			     5.调用库函数，初始化GPIO
		 
       
        <main.c>:1.调用LED的GPIO的初始化函数
	        2.while里面为LED的颜色状态变化循环
